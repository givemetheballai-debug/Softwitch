# SoftWitch

A modern cosmic universe for self-reflection, intuition, and gentle rituals. Witchcraft for normal people.

## Features

- **Birth Chart Calculator**: Calculate your sun, moon, and rising signs with personalized mantras and rituals
- **Lore Library**: Explore rituals, mantras, and cosmic wisdom
- **Daily Grounding**: See today's moon phase and seasonal energy
- **Cosmic Projects**: Connect to the full universe of tools (Cosmic MASH, Big Scream Energy, etc.)

## Tech Stack

- React + Vite
- Tailwind CSS
- Framer Motion (animations)
- React Router (navigation)
- SunCalc (moon phase calculations)

## Local Development

```bash
npm install
npm run dev
```

## Deployment

Built for Vercel deployment via GitHub.

```bash
npm run build
```

## Philosophy

SoftWitch combines product strategy with intuition, modern design with gentle magic, and AI-assisted building with human intention. Everything here is an invitation, not a prescription. Magic is pattern recognition with better lighting.
